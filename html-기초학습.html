01

HTML이란 무엇인가?

HTML은 HyperText Markup Language의 약자입니다.

웹페이지의 뼈대(구조)를 만드는 언어입니다.

쉽게 말하면, 브라우저(크롬, 엣지 등)에게 "여기는 제목이야", "여기는 목록이야" 라고 알려주는 언어입니다.

02

태그(Tag)란?

HTML은 태그라는 단위로 이루어져 있습니다.

태그는 꺾쇠괄호(< >)로 감싸며, 대부분 여는 태그와 닫는 태그 한 쌍으로 이루어집니다.

예시:

  • <h1> 제목입니다 </h1>
  • <p> 문단입니다 </p>
  • <b> 굵게 </b>

닫는 태그는 슬래시(/)가 앞에 붙습니다: </태그명>

혼자 쓰는 태그도 있습니다 (닫는 태그 없음): <hr>, <br>

03

HTML 문서의 기본 구조

모든 HTML 파일은 아래 구조를 기본으로 가집니다:

<!DOCTYPE html>         ← 이 파일이 HTML임을 선언
<html lang="ko">        ← HTML 문서 시작
  <head>               ← 보이지 않는 설정 영역
    <meta charset="UTF-8">  ← 한글 깨짐 방지
    <title>탭 제목</title>   ← 브라우저 탭에 표시되는 이름
  </head>

  <body>               ← 실제 화면에 보이는 영역
    여기에 내용 작성!
  </body>
</html>               ← HTML 문서 끝
04

제목 태그 <h1> ~ <h6>

h 뒤의 숫자가 클수록 제목이 작아집니다.

h1 - 가장 큰 제목

h2 - 두 번째 제목

h3 - 세 번째 제목

h4 - 네 번째 제목

h5 - 다섯 번째 제목
h6 - 가장 작은 제목

보통 페이지에서 h1은 한 번만, h2~h3은 섹션 제목으로 씁니다.

05

문단 태그 <p> 와 줄바꿈 <br>

이것은 첫 번째 문단입니다. p 태그는 위아래로 여백이 생깁니다.

이것은 두 번째 문단입니다. 글을 쓸 때 가장 자주 쓰는 태그입니다.

br 태그는 줄바꿈만 합니다:
이렇게 같은 문단 안에서 줄을 바꿀 수 있습니다.
여백 없이 바로 다음 줄로 넘어갑니다.

06

텍스트 강조

일반 텍스트입니다.

b 태그: 굵게(Bold) 표시됩니다.

i 태그: 기울임(Italic) 표시됩니다.

u 태그: 밑줄(Underline) 표시됩니다.

s 태그: 취소선 표시됩니다.

조합도 가능합니다: 굵게 + 기울임

텍스트 안에서 일부분만 강조할 수도 있습니다.

07

목록 태그

순서 없는 목록 <ul> (점으로 표시)

  • 사과
  • 바나나
  • 포도

순서 있는 목록 <ol> (번호로 표시)

  1. 먼저 이것을 하세요
  2. 그 다음 이것을 하세요
  3. 마지막으로 이것을 하세요

ul = unordered list (순서 없음), ol = ordered list (순서 있음)

li = list item (목록 항목)

08

링크 태그 <a>

a 태그는 클릭하면 다른 페이지로 이동하는 링크를 만듭니다.

href 속성에 이동할 주소를 씁니다.

예시1 (현재 탭에서 열림): 네이버 바로가기

예시2 (새 탭에서 열림): 구글 새 탭으로 열기

예시3 (이메일): 이메일 보내기

target="_blank" 를 추가하면 새 탭에서 열립니다.

09

이미지 태그 <img>

img 태그로 이미지를 삽입합니다. 닫는 태그가 없습니다.

src 속성에 이미지 주소를, alt 속성에 이미지 설명을 씁니다.

alt는 이미지가 안 보일 때 대신 표시되는 텍스트입니다.

사용법: <img src="이미지주소" alt="이미지설명">

예시:
<img src="dog.jpg" alt="강아지 사진">         ← 같은 폴더의 파일
<img src="https://..." alt="인터넷 이미지">   ← 인터넷 주소
10

표(Table) 태그

표를 만드는 태그입니다.

  • table : 표 전체
  • tr : 표의 행(가로줄) — table row
  • th : 표의 제목 칸 (굵게 표시됨) — table header
  • td : 표의 일반 칸 — table data
이름나이직업
홍길동25학생
김철수30개발자
11

주석(Comment)

주석은 코드 안에 메모를 남기는 것입니다.

브라우저 화면에는 표시되지 않습니다.

코드를 설명하거나, 잠깐 숨기고 싶을 때 사용합니다.

사용법: <!-- 여기에 메모를 씁니다 -->

예시:
<!-- 이 아래는 소개 섹션입니다 -->
<h2>소개</h2>
12

자주 하는 실수

  1. 닫는 태그를 빠뜨림 — <p> 를 열었으면 반드시 </p> 로 닫아야 합니다
  2. 태그 철자 오류 — <tabel> (X) → <table> (O)
  3. 한글 깨짐 — <meta charset="UTF-8"> 이 없으면 한글이 깨질 수 있습니다
  4. 파일 저장 안 함 — 수정 후 반드시 저장(Ctrl+S) 후 브라우저에서 새로고침(F5) 하세요
